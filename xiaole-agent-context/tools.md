# 后端可调用工具

1. task_manager       管理任务状态与推进
2. scheduler          调度定时任务
3. memory_engine      处理短期和长期记忆
4. notify             推送消息/提醒用户
5. database           访问数据库接口
6. deploy_scripts     部署相关操作
7. system_info        获取系统状态信息
8. logging            记录日志、调试信息

> 使用这些工具必须遵守 dev-rules.md 和 task-basic-rules.md
